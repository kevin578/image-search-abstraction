Image Search Abstraction

Usage:

https://kevins-image-search.herokuapp.com/imagesearch/item to be searched

Returns a list of 10 search items.

https://kevins-image-search.herokuapp.com/imagesearch/item to be searched/offset=3

Paginates the list. In this case it would go to the 3rd page.

https://kevins-image-search.herokuapp.com/searches/

Returns previous 10 searches.
